Implementation of LinGO, an AR language learning app, for COMS4170 User Interface Design at Columbia University.

Setup

1. Clone the repo.
2. Install following dependencies if needed:
      "dependencies": {
        "@testing-library/jest-dom": "^5.14.1",
        "@testing-library/react": "^13.0.0",
        "@testing-library/user-event": "^13.2.1",
        "@types/jest": "^27.0.1",
        "@types/node": "^16.7.13",
        "@types/react": "^18.0.0",
        "@types/react-dom": "^18.0.0",
        "react": "^18.2.0",
        "react-dom": "^18.2.0",
        "react-router-dom": "^6.5.0",
        "react-scripts": "5.0.1",
        "typescript": "^4.4.2"
      }
3. Run npm start in terminal inside project folder.
4. Open http://localhost:3000 to view it in the browser.
